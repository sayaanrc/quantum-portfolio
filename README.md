# Quantum Computing Portfolio

This repository organizes my work in quantum computing, focusing on core algorithms, applied optimization and physics-driven modelling. The listed projects highlight both theoretical understanding and practical implementation experience using Qiskit and Python.

  Repository: https://github.com/sayaanrc/quantum-battery-optimization
